#  Unity 3D Project Skeleton Revenge 骷髅复仇记
# 简介
所设计的闯关游戏《骷髅复仇记（Skeleton Revenge）》，是一款通关时间较短的闯关游戏，支持键盘和手柄输入，提供两个关卡供游玩，拥有极好的角色操控手感！
# 操作
操作|	键盘|	手柄|
-|-|-|
移动|	WASD|	L|
加速|	Shift|	LB|
跳跃	|空格|	Y
旋转视角|鼠标|	R
